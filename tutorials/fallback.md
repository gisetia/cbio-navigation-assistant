# cBioPortal Tutorials
Use this quick reference when no page-specific instructions are available; it outlines the common workflow and troubleshooting reminders.

Focus each exploration on a clear cohort, iterate between cohort-level summaries and patient-level evidence, and watch for system messages that explain why certain charts or statistics may be unavailable. Use the download/share controls on each view to capture progress before switching tabs or running a different query.
